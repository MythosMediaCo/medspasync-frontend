## 🚀 **Usage Instructions**

### **Generate Demo Data**
```bash
# Urban scenario (12 months of data)
npm run demo:urban

# Enterprise scenario (24 months of data)
npm run demo:enterprise

# All scenarios
npm run demo:all
```

### **Run Live Demo**
```bash
# Urban scenario demo
npm run demo:run:urban

# Enterprise scenario demo
npm run demo:run:enterprise

# Interactive demo
npm run demo:run:interactive
```
